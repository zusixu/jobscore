## 🚀 部署指南

### 前置要求
- Node.js 12.0 以上
- npm 6.0 以上（或 yarn）

### 快速部署步骤

```bash
# 1. 安装依赖
npm install

# 2. 本地开发测试
npm run dev
# 在浏览器打开 http://localhost:5173

# 3. 生产环境构建
npm run build
# 生成 dist 文件夹

# 4. 上传到服务器
# 将 dist 文件夹中的所有文件上传到服务器的 public_html 或相应目录

# 5. 预览生产构建
npm run serve
```

- **Tailwind CSS** - 响应式样式框架
- **Element Plus** - UI 组件库

### 响应式设计
- **CSS Grid** - 多断点布局
- **Flexbox** - 组件内部布局
- **CSS Media Queries** - 响应式样式调整
- **Tailwind Breakpoints** - sm, md, lg, xl

### 交互效果
- **CSS Transitions** - 平滑过渡
- **CSS Transforms** - 变形动画
- **Vue State Management** - 状态驱动 UI
